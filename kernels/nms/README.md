# NMS

## 0x00 说明

包含以下内容：

- [X] nms_kernel(CPU/GPU)
- [X] PyTorch bindings

nms cuda实现是最基础的版本，根据[官方源码](https://github.com/pytorch/vision/blob/main/torchvision/csrc/ops/cuda/nms_kernel.cu)可以进行进一步优化。
